发布说明

本文档记录了 UTags 的版本更新历史，包括各个版本的主要功能和改进。

v0.20.x​

  • 添加 Shadow DOM 遍历支持
  • 优化 bilibili.com 网站的标签功能
  • 为 pornhub.com 网站添加标签功能
  • 为 youtube.com 网站添加特殊标签过滤和快速星标功能
  • 优化用户名匹配逻辑

v0.19.x​

  • 添加 WebDAV 同步支持
  • 改进标签建议算法
  • 添加标签层次结构支持
  • 优化移动设备上的触摸支持

v0.18.x​

  • 添加深色模式支持
  • 改进数据导入/导出功能
  • 添加多语言支持
  • 优化性能和内存使用

v0.17.x 及更早版本​

  • 初始版本发布
  • 基本标签功能实现
  • GitHub 同步支持
  • 特殊标签功能（block、hide、star）

完整的版本历史和更新日志请参考 GitHub 仓库 中的发布记录。
